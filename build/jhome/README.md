## This little ditty uses sets the java_home based on /usr/libexec/java_home to the local shell.

Usage:
set JAVA_HOME to latest java
~~~
jhome 
~~~
set JAVA_HOME to a certain java version
~~~
jhome -v 1.7
~~~
